Who Am I?

I'm Rehan — a 17-year-old tech enthusiast from Indonesia with an insatiable curiosity for how things work. Since childhood, I've been fascinated by computers, networks, and the logic behind every line of code.
As a Network Engineer, I specialize in designing, implementing, and securing robust network infrastructures. From configuring Cisco devices to analyzing packet flows with Wireshark, I ensure seamless connectivity and optimal performance.
As a Web Developer, I craft responsive, user-centric web applications using modern technologies like React, Node.js, and various CSS frameworks. I enjoy bridging the gap between backend logic and frontend aesthetics, creating experiences that are both functional and visually engaging.
